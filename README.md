# proyecto-simulacion-powerdevs
Proyecto para la materia Simulacion (Codigo 1962) de la Universidad Nacional de Rio Cuarto, Argentina.

**Se recomienda leer las primeras paginas del informe antes de ejecutar el proyecto.**

#### Instrucciones para ejecutar el modelo
1. Descargar la version dockerizada de powerdevs utilizada para el proyecto. [Link al repositorio.] (https://git.exactas.uba.ar/sed/powerdevs-docker.git)
2. Copiar la carpeta src de este repositorio a la carpeta atomics de powerdevs y renombrarla a proyecto-simulacion-powerdevs.
